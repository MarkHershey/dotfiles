## Ubuntu Setup

Desktop Apps:

-   Brave Browser
    ```bash
    sudo snap install brave
    ```
-   Telegram Desktop
    ```bash
    sudo snap install telegram-desktop
    ```
-   Spotify
    ```bash
    sudo snap install spotify
    ```
-   Visual Studio Code
    ```bash
    sudo snap install code --classic
    ```
-   htop
    ```bash
    sudo snap install htop
    ```
-   tweaks
    ```bash
    sudo apt install gnome-tweaks
    ```
